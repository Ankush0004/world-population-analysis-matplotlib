# 🌍 World Population Analysis

Exploratory data analysis and visualization of world population data by country and gender using Python.

## 📊 Project Overview

This project analyzes global population trends using the World Population dataset. It focuses on identifying the top and bottom countries by population and breaks down the data by gender using stacked bar charts.

## 📸 Sample Output

> Stacked bar chart showing Bottom 10 Countries by Total Population (2022) — split by Male and Female population.

## 🛠️ Tech Stack

- **Python**
- **Pandas** — data loading and manipulation
- **NumPy** — numerical operations
- **Matplotlib** — data visualization

## 📁 Dataset

- World Population Dataset (country-wise, year-wise, gender-wise)

## ⚙️ How to Run

1. Clone the repository
```bash
git clone https://github.com/Ankush0004/world-population-analysis-matplotlib.git
cd world-population-analysis-matplotlib
```

2. Install dependencies
```bash
pip install pandas numpy matplotlib
```

3. Run the notebook
```bash
jupyter notebook task1.ipynb
```

## 📌 Key Insights

- Identified the bottom 10 countries by total population in 2022
- Visualized gender distribution (Male vs Female) using stacked bar charts
- Compared population across countries using grouped bar plots
